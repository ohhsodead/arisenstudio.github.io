Got Some Questions?

HERE ARE SOME ANSWERS FOR YOU

Where can I download the latest version of this project?

Our official releases can be found only on GitHub/SourceForge. You can download the latest installer here.

Arisen Studio failed to download, what do I do?

This is generally caused by an antivirus or firewall blocking files. Add Arisen Studio to your exception list and retry the installation.

Why did my antivirus flag Arisen Studio?

This is generally because our source code isn't protected and/or signed with Windows. It's a false positive. Arisen Studio is completely open source which means you can also check out the code for anything you suspect might be malicious.

When is “_” going to be updated or released?

We cannot provide an ETA of when a mod will be released or updated. We only source them. If any are outdated please let us know!

I've found a bug, what should I do?

If this is a bug with the tool not functioning correctly (e.g. downloads or installs not working) then please report it to us. You can open a ticket in #support channel in our Discord server to get help from one of our team. Note: If your problem is with the mods then read the next answer.

I've found a bug with specific mods, where can I report this?

Any issues with mods (e.g. function not working) should be reported to the appropriate creator/owner as we can't do anything about this. If a new version is released then we'll add that version to our database. You can also check out the #modding-help channel in our Discord server where other modders could be able to help.

I would like to request a feature, where can I do this?

We love to hear your ideas to help make this project even better! Please let us know in the #suggestions channel in our Discord server so we can discuss it before going further.
